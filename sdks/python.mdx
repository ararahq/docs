## Instalação e setup

```python
from arara_api_sdk import AraraClient

# Síncrono
client = AraraClient(
    api_key="ara_live_xxx",
    base_url="https://api.ararahq.com",
)

# Com context manager (fecha conexões automaticamente)
with AraraClient(api_key="ara_live_xxx") as client:
    # uso aqui
    ...
```

## Enviar mensagem

```python
from arara_api_sdk import AraraClient
from arara_api_sdk.models.message import SendMessageRequest

with AraraClient(api_key="ara_live_xxx") as client:
    response = client.messages.send(SendMessageRequest(
        receiver="whatsapp:+5511999999999",
        template_name="confirmacao_pedido",
        template_variables=[
            "https://meusite.com/nota/123.pdf",  # header (mídia opcional)
            "João",                                # {{1}}
            "ARA-12345",                           # {{2}}
        ],
    ))

print(response.id)      # ara_msg_xxx
print(response.status)  # PENDING
```

**Mensagem de sessão (texto livre, dentro de janela 24h):**

```python
response = client.messages.send(SendMessageRequest(
    receiver="whatsapp:+5511999999999",
    body="Oi! Vi sua dúvida e respondo já já.",
))
```

## Async / asyncio

Toda chamada tem variante `_async`:

```python
import asyncio
from arara_api_sdk import AraraClient
from arara_api_sdk.models.message import SendMessageRequest

async def main():
    async with AraraClient(api_key="ara_live_xxx") as client:
        responses = await asyncio.gather(*[
            client.messages.send_async(SendMessageRequest(
                receiver=f"whatsapp:+551199999{i:04d}",
                template_name="lembrete_aula",
                template_variables=["João"],
            ))
            for i in range(100)
        ])
        return responses

results = asyncio.run(main())
```

Bom pra batch sem `messages/batch` (ainda) ou quando precisa intercalar lógica entre envios.

## Templates

```python
from arara_api_sdk.models.template import CreateTemplateRequest

# Listar
templates = client.templates.list()

# Criar
template = client.templates.create(CreateTemplateRequest(
    name="promo_natal",
    category="MARKETING",
    language="pt_BR",
    body="Oi {{1}}, oferta de Natal: {{2}}% off até {{3}}.",
    samples=["João", "30", "sexta"],
))

# Status
status = client.templates.get_status("promo_natal")
print(status.status)  # PENDING | APPROVED | REJECTED | PAUSED
```

## Campanhas

```python
from arara_api_sdk.models.campaign import CreateCampaignRequest

campaign = client.campaigns.create(CreateCampaignRequest(
    name="blackfriday-2026",
    template_name="bf_oferta_geral",
    contacts=[
        {"phone": "+5511999999999", "variables": {"name": "João", "discount": "30"}},
        {"phone": "+5511888888888", "variables": {"name": "Maria", "discount": "25"}},
    ],
))

# Acompanha
detail = client.campaigns.get(campaign.id)
print(detail.status)  # PROCESSING | COMPLETED | CANCELLED
```

## Pydantic models

Todos os request/response são Pydantic models. Você ganha validation grátis:

```python
from arara_api_sdk.models.message import SendMessageRequest
from pydantic import ValidationError

try:
    req = SendMessageRequest(
        receiver="invalid",  # vai falhar — deve ter prefixo whatsapp:
        body="oi",
    )
except ValidationError as e:
    print(e.errors())
```

E exportar como dict/JSON pra logs:

```python
req.model_dump_json(indent=2)
```

## Idempotency

POSTs que criam recurso aceitam `idempotency_key`:

```python
import uuid

client.messages.send(SendMessageRequest(...), idempotency_key=str(uuid.uuid4()))
```

## Erros

```python
from arara_api_sdk.exceptions import (
    AraraError,
    AraraAuthError,
    AraraValidationError,
    AraraRateLimitError,
    AraraResourceNotFoundError,
    AraraServerError,
    AraraConnectionError,
    AraraTimeoutError,
)

try:
    client.messages.send(...)
except AraraRateLimitError as e:
    time.sleep(e.retry_after)
except AraraError as e:
    print(e.code, e.message, e.status_code)
```

## Retry com tenacity

O SDK usa `tenacity` por baixo. Retry default em 429/5xx com backoff exponencial. Configurável:

```python
client = AraraClient(
    api_key="...",
    max_retries=5,
    retry_max_wait=30,
)
```
