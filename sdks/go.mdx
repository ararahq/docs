```bash
go get github.com/ararahq/arara-go-sdk
```

<Note>
  **Beta.** O SDK cobre todo o core da API. Publicação de tag estável (`v1.0.0` no `pkg.go.dev`) em andamento. Enquanto isso, `go get` puxa a versão do repositório.
</Note>

Zero dependências externas — só a stdlib (`net/http`, `encoding/json`, `context`). Go 1.22+.

## Setup

```go
import "github.com/ararahq/arara-go-sdk"

client := arara.NewClient(os.Getenv("ARARA_API_KEY"))
```

A API key usa o prefixo `ara_live_` (produção — a Arara não tem modo de teste). Opções via variadic:

```go
client := arara.NewClient(
    os.Getenv("ARARA_API_KEY"),
    arara.WithBaseURL("https://api.ararahq.com"), // default
    arara.WithTimeout(10*time.Second),            // default 10s
)
```

Todo método recebe `context.Context` como primeiro argumento. Retry com backoff exponencial em 429/5xx honrando `Retry-After` vem embutido.

## Enviar um template

```go
resp, err := client.Messages.Send(ctx, arara.SendMessageRequest{
    Receiver:     "whatsapp:+5511999999999",
    TemplateName: "fatura_disponivel",
    TemplateVariables: []string{
        "https://meusite.com/fatura.pdf", // header (mídia)
        "João",                            // {{1}}
        "Janeiro",                         // {{2}}
    },
})
if err != nil {
    log.Fatal(err)
}
fmt.Println(resp.ID)
```

**Padrão de mídia**: a URL da mídia vai como **primeiro elemento** de `TemplateVariables`. As posições seguintes preenchem `{{1}}`, `{{2}}`... do corpo do template.

## Erros

Erros da API vêm como `*arara.Error`, com o envelope estruturado desserializado:

```go
resp, err := client.Messages.Send(ctx, req)
if err != nil {
    var apiErr *arara.Error
    if errors.As(err, &apiErr) {
        fmt.Println(apiErr.StatusCode, apiErr.Code, apiErr.Message)
        if apiErr.RetryAfter > 0 {
            // rate limit — o cliente já tentou o retry automático
        }
    }
}
```

## Escopo da API key

Keys `READ` só fazem GET; keys `SEND` fazem GET + envio (`/messages`, `/campaigns`); o resto (criar campanha, editar número, girar api-key) exige key `ADMIN`.
