# Cookbook: Recuperação de Carrinho Abandonado

O abandono de carrinho é um dos maiores desafios do e-commerce. O WhatsApp Arara permite que você re-engaje o cliente com uma mensagem personalizada e um link direto para o checkout, muitas vezes com um incentivo extra (cupom).

## Estratégia de Disparo

Recomendamos um fluxo de 2 etapas:
1.  **30 minutos após o abandono**: Uma mensagem amigável perguntando se houve algum problema técnico.
2.  **24 horas após o abandono**: Uma oferta especial (ex: 5% OFF) válida por tempo limitado.

---

## Criando o Template

Use o dashboard para criar um template que aceite as seguintes variáveis:
- `{{1}}`: Nome do cliente.
- `{{2}}`: Nome do produto ou quantidade de itens.
- `{{3}}`: Link de checkout (idealmente um **Smart Link**).

**Exemplo:**
> Olá {{1}}! Vimos que você esqueceu {{2}} no seu carrinho. Para te ajudar a decidir, use o cupom **VOLTEI** e finalize sua compra aqui: {{3}}

---

## Exemplo de Implementação (Node.js)

```javascript
const recoverCart = async (customerData) => {
  const { phone, name, itemsCount, checkoutUrl } = customerData;
  
  await arara.messages.send({
    receiver: `whatsapp:${phone}`,
    templateName: 'cart_recovery_main',
    variables: [
      name,
      `${itemsCount} itens`,
      checkoutUrl
    ]
  });
};
```

## Dicas de Alta Conversão

### 1. Timing é Tudo
Não espere demais. O interesse do cliente esfria rapidamente. Disparar o primeiro contato em menos de 1 hora aumenta as chances de conversão em até 4x.

### 2. Use a "Escassez"
No segundo disparo (24h depois), use frases como *"Reservamos seu estoque por mais 2 horas"* para criar urgência.

### 3. Smart Links + Webhooks
Use **Smart Links** no botão de checkout. Se o cliente clicar mas não comprar, seu sistema recebe o evento `link.clicked` e você pode tentar uma abordagem de suporte humano via chat.
