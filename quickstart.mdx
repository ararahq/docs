Integre a Arara e envie sua primeira mensagem via WhatsApp em menos de 2 minutos. Projetada por desenvolvedores, para desenvolvedores.

## Jato da Decolagem

Antes de começar, certifique-se de ter:
- **Uma conta ativa**: Faça login ou crie a sua no [Dashboard](https://www.ararahq.com/dashboard).
- **Sua API Key**: Gerada instantaneamente no primeiro acesso.
- **Linha Pronta**: Seu número dedicado configurado (caso ainda não tenha, fale com o suporte).

## Tutorial: Sua Primeira Mensagem 🚀

<Steps>
  ### Passo 1: Capture sua Chave de API
  <Step title="Acesse suas Credenciais de Elite">
    O acesso à Arara é protegido por uma chave única. Ao realizar seu primeiro login no [Dashboard](https://www.ararahq.com/dashboard), sua chave será gerada e exibida.

    <Warning>
      **IMPORTANTE**: Guarde sua chave imediatamente! Você não poderá vê-la novamente depois que fechar a tela. A chave terá o formato: `ara_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
    </Warning>

    <Info>
      Sua API Key deve ser tratada como uma senha. **Nunca a exponha em código cliente ou repositórios públicos.**
    </Info>
  </Step>

  ### Passo 2: Crie um Template
  <Step title="Crie um Template no Dashboard">
    No dashboard, navegue até a seção **"Templates"** e crie um novo template. Por exemplo:

    - **Nome**: `template_teste_inicio_conversa`
    - **Conteúdo**: `Olá! Esta é uma mensagem de teste da Arara.`

    <Warning>
      **Importante**: Você só pode enviar mensagens usando templates que foram criados previamente no dashboard.
    </Warning>
  </Step>

  ### Passo 3: Faça a Chamada da API
  <Step title="Envie a Mensagem com cURL">
    Abra seu terminal ou cliente HTTP e faça uma requisição `POST` para o endpoint de envio de mensagens.

    Substitua:
    -   `<SUA_API_KEY>` pela sua chave de API (`ara_sk_***`).
    -   `<NUMERO_DESTINATARIO>` pelo número no formato `whatsapp:+5583991768778`.
    -   `<NOME_TEMPLATE>` pelo nome do template que você criou.

    <CodeGroup>
    ```bash cURL (Template)
    curl -X POST https://api.ararahq.com/api/v1/messages \
         -H "Authorization: Bearer <SUA_API_KEY>" \
         -H "Content-Type: application/json" \
         -d '{
               "receiver": "whatsapp:+5583991768778",
               "templateName": "template_teste_inicio_conversa",
               "variables": []
             }'
    ```

    ```bash cURL (Mídia)
    curl -X POST https://api.ararahq.com/api/v1/messages \
         -H "Authorization: Bearer <SUA_API_KEY>" \
         -H "Content-Type: application/json" \
         -d '{
               "receiver": "whatsapp:+5583991768778",
               "templateName": "mensagem_com_imagem",
               "variables": ["Micael"],
               "media_url": "https://ararahq.com/l/FtFmja"
             }'
    ```

    ```python Python
    import requests

    api_key = "<SUA_API_KEY>"
    receiver = "whatsapp:+5583991768778"
    
    headers = {
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json"
    }
    
    payload = {
        "receiver": receiver,
        "templateName": "template_teste_inicio_conversa",
        "variables": []
    }
    
    response = requests.post("https://api.ararahq.com/api/v1/messages", headers=headers, json=payload)
    
    print(response.json())
    ```

    ```java Java
    import java.net.http.HttpClient;
    import java.net.http.HttpRequest;
    import java.net.http.HttpResponse;
    import java.net.URI;
    import java.time.Duration;
    import com.fasterxml.jackson.databind.ObjectMapper;

    HttpClient client = HttpClient.newBuilder()
        .connectTimeout(Duration.ofSeconds(10))
        .build();

    ObjectMapper mapper = new ObjectMapper();
    String jsonBody = mapper.writeValueAsString(Map.of(
        "receiver", "whatsapp:+5583991768778",
        "templateName", "template_teste_inicio_conversa",
        "variables", List.of()
    ));

    HttpRequest request = HttpRequest.newBuilder()
        .uri(URI.create("https://api.ararahq.com/api/v1/messages"))
        .header("Authorization", "Bearer <SUA_API_KEY>")
        .header("Content-Type", "application/json")
        .POST(HttpRequest.BodyPublishers.ofString(jsonBody))
        .build();

    HttpResponse<String> response = client.send(request, 
        HttpResponse.BodyHandlers.ofString());
    System.out.println(response.body());
    ```

    ```kotlin Kotlin
    import kotlinx.serialization.Serializable
    import kotlinx.serialization.encodeToString
    import kotlinx.serialization.json.Json
    import java.net.http.HttpClient
    import java.net.http.HttpRequest
    import java.net.http.HttpResponse
    import java.net.URI
    import java.time.Duration

    @Serializable
    data class MessageRequest(
        val receiver: String,
        val templateName: String,
        val variables: List<String> = emptyList()
    )

    val client = HttpClient.newBuilder()
        .connectTimeout(Duration.ofSeconds(10))
        .build()

    val requestBody = MessageRequest(
        receiver = "whatsapp:+5583991768778",
        templateName = "template_teste_inicio_conversa",
        variables = emptyList()
    )

    val jsonBody = Json.encodeToString(requestBody)

    val request = HttpRequest.newBuilder()
        .uri(URI.create("https://api.ararahq.com/api/v1/messages"))
        .header("Authorization", "Bearer <SUA_API_KEY>")
        .header("Content-Type", "application/json")
        .POST(HttpRequest.BodyPublishers.ofString(jsonBody))
        .build()

    val response = client.send(request, HttpResponse.BodyHandlers.ofString())
    println(response.body())
    ```
    </CodeGroup>
  </Step>

  ### Passo 4: Verifique a Resposta
  <Step title="Confirme o Envio">
    Se a requisição for bem-sucedida, você receberá uma resposta `202 Accepted` com JSON similar a esta:
    ```json
    {
      "id": "ara_msg_k9Bf2XqP",
      "receiver": "whatsapp:+5583991768778",
      "templateName": "template_teste_inicio_conversa",
      "status": "PENDING",
      "messageType": "TEMPLATE",
      "createdAt": "2024-01-15T10:30:00Z"
    }
    ```
    
    A mensagem será processada e enviada em breve. Você pode usar o `id` retornado para consultar o status futuramente. Verifique o histórico completo no seu [Dashboard](https://www.ararahq.com/dashboard).
  </Step>
</Steps>
