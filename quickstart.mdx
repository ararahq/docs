Você vai mandar sua primeira mensagem WhatsApp em **5 minutos**. Use o modo **TEST** primeiro (gratuito, sem cobrança, sem aprovação Meta) e depois passa pra produção.

<Note>
  Pra entender **antes** os conceitos por trás (janela 24h, templates HSM, pricing): leia [Como funciona a WhatsApp Business API](/concepts/whatsapp). 10 minutos que economizam dias.
</Note>

## 1. Crie sua conta e pegue a API key (1 min)

Acesse o [Dashboard](https://ararahq.com/dashboard) e crie conta com Google ou email. No primeiro login a Arara gera:

- 1 chave `ara_test_*` (modo TEST — gratuita, ilimitada, simula envios)
- 1 chave `ara_live_*` (modo LIVE — produção, consome créditos)

A chave **completa só aparece uma vez**. Copie e guarde em variável de ambiente:

```bash
export ARARA_API_KEY=ara_test_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

<Warning>
  Nunca commite a key. Nunca exponha no frontend. Use env var, Vercel/Netlify secrets, AWS Secrets Manager.
</Warning>

## 2. Mande a primeira mensagem (3 min)

Em modo TEST, **qualquer número é válido** — a Arara simula o envio. Use seu próprio número pra ver na prática:

<CodeGroup>
```bash cURL
curl -X POST https://api.ararahq.com/api/v1/messages \
  -H "Authorization: Bearer $ARARA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "receiver": "whatsapp:+5511999999999",
    "templateName": "hello_world",
    "templateVariables": ["seu_nome"]
  }'
```

```typescript Node.js
// npm install @ararahq/sdk
import { NodeSDK } from '@ararahq/sdk';

const sdk = new NodeSDK({
  apiKey: process.env.ARARA_API_KEY!,
  baseUrl: 'https://api.ararahq.com/api',
});

const response = await sdk.messages.send({
  receiver: 'whatsapp:+5511999999999',
  templateName: 'hello_world',
  templateVariables: ['seu_nome'],
});

console.log(response.id, response.status);
```

```python Python
# pip install ararahq-sdk
import os
from arara_api_sdk import AraraClient
from arara_api_sdk.models.message import SendMessageRequest

with AraraClient(api_key=os.environ["ARARA_API_KEY"]) as client:
    response = client.messages.send(SendMessageRequest(
        receiver="whatsapp:+5511999999999",
        template_name="hello_world",
        template_variables=["seu_nome"],
    ))

print(response.id, response.status)
```

```php PHP
// composer require ararahq/sdk
use Arara\Arara;
use Arara\Config;

$sdk = new Arara(new Config(apiKey: getenv('ARARA_API_KEY')));

$response = $sdk->messages->send(
    receiver: 'whatsapp:+5511999999999',
    templateName: 'hello_world',
    variables: ['seu_nome'],
);

echo "{$response->id} {$response->status}";
```

```java Java
// implementation("com.ararahq:arara-java-sdk:1.8.0")
import com.ararahq.arara.sdk.Arara;
import com.ararahq.arara.sdk.models.SendMessageRequest;
import java.util.List;

Arara arara = Arara.builder()
    .apiKey(System.getenv("ARARA_API_KEY"))
    .build();

var response = arara.getMessages().send(
    SendMessageRequest.builder()
        .receiver("whatsapp:+5511999999999")
        .templateName("hello_world")
        .templateVariables(List.of("seu_nome"))
        .build()
);

System.out.println(response.getId() + " " + response.getStatus());
```

```go Go
// go install (SDK Go em breve — usando net/http direto)
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
    "os"
)

func main() {
    body, _ := json.Marshal(map[string]any{
        "receiver":          "whatsapp:+5511999999999",
        "templateName":      "hello_world",
        "templateVariables": []string{"seu_nome"},
    })
    req, _ := http.NewRequest("POST",
        "https://api.ararahq.com/api/v1/messages",
        bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("ARARA_API_KEY"))
    req.Header.Set("Content-Type", "application/json")
    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    fmt.Println(resp.Status)
}
```
</CodeGroup>

Response esperada (`202 Accepted`):

```json
{
  "id": "ara_msg_k9Bf2XqP",
  "status": "queued",
  "mode": "TEST",
  "sender": "whatsapp:+551144440000",
  "receiver": "whatsapp:+5511999999999"
}
```

`status` começa em `queued`, vai pra `sent`, `delivered` (e `read` quando cliente abre).

## Modo TEST → modo LIVE

Antes de ir pra produção:

<Steps>
  <Step title="Crie template real">
    `hello_world` é só pra teste. Crie seus templates em [Templates HSM](/concepts/templates). Aprovação Meta: 1-72h.
  </Step>
  <Step title="Configure webhook">
    Receba status (`sent`, `delivered`, `read`, `failed`) em tempo real. Ver [Webhooks](/webhooks).
  </Step>
  <Step title="Troca pra chave LIVE">
    Substitua `ara_test_*` por `ara_live_*` na env var. Tudo igual no código.
  </Step>
  <Step title="Liga auto-recarga">
    Saldo de créditos zerado interrompe envios. Ver [Pricing](/concepts/pricing) e [auto-recharge](/recovery).
  </Step>
</Steps>

## Erros comuns

| Sintoma | Causa provável | Fix |
|---|---|---|
| `401 INVALID_API_KEY` | Chave errada ou modo trocado (test vs live) | Cheque `ARARA_API_KEY` |
| `404 TEMPLATE_NOT_FOUND` | Template não existe ou não aprovado | Use `hello_world` em test, ou crie um seu |
| `422 OUTSIDE_SESSION_WINDOW` | Tentou texto livre sem cliente ter respondido em 24h | Use template HSM. Ver [Janela 24h](/concepts/24h-window) |
| `402 INSUFFICIENT_CREDITS` | Modo LIVE sem saldo | Recarrega via dashboard ou [auto-recharge](/recovery) |

Lista canônica em [Erros](/erros).
