Integre a Arara e envie sua primeira mensagem em menos de 2 minutos.

## 1. Obtenha sua API Key

Acesse o [Dashboard](https://www.ararahq.com/dashboard) e copie sua chave de API. Ela tem o formato:

```
ara_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

<Warning>
  A chave e exibida apenas uma vez no primeiro login. Guarde-a em um local seguro.
</Warning>

## 2. Envie sua primeira mensagem

<CodeGroup>

```bash cURL
curl -X POST https://new-api.ararahq.com/api/v2/messages \
  -H "Authorization: Bearer ara_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "5511999999999",
    "channel": "whatsapp",
    "template": "hello_world_v2"
  }'
```

```javascript Node.js
const response = await fetch("https://new-api.ararahq.com/api/v2/messages", {
  method: "POST",
  headers: {
    "Authorization": "Bearer ara_live_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    to: "5511999999999",
    channel: "whatsapp",
    template: "hello_world_v2",
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://new-api.ararahq.com/api/v2/messages",
    headers={
        "Authorization": "Bearer ara_live_xxx",
        "Content-Type": "application/json",
    },
    json={
        "to": "5511999999999",
        "channel": "whatsapp",
        "template": "hello_world_v2",
    },
)

print(response.json())
```

</CodeGroup>

## 3. Verifique a resposta

Se tudo deu certo, voce recebe `202 Accepted`:

```json
{
  "id": "ara_msg_k9Bf2XqP",
  "to": "5511999999999",
  "channel": "whatsapp",
  "template": "hello_world_v2",
  "status": "QUEUED",
  "createdAt": "2026-04-04T14:30:00Z"
}
```

O campo `status` comeca como `QUEUED` e evolui para `SENT`, `DELIVERED` ou `READ` conforme o WhatsApp processa. Use o `id` para consultar o status a qualquer momento:

```bash
curl https://new-api.ararahq.com/api/v2/messages/ara_msg_k9Bf2XqP \
  -H "Authorization: Bearer ara_live_xxx"
```

<Accordion title="Migrando da V1?">
  A V1 continua funcionando em `https://api.ararahq.com/api/v1`. As principais mudancas na V2:

  - O campo `receiver` agora e `to` (sem o prefixo `whatsapp:+`)
  - O campo `templateName` agora e `template`
  - Novo campo obrigatorio `channel` (`whatsapp` ou `email`)
  - Base URL: `https://new-api.ararahq.com/api`

  Consulte a [referencia da API V1](/api-reference/endpoint/messages) para a documentacao legada.
</Accordion>
