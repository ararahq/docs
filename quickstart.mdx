## Boas-vindas ao Mundo da Comunicação Simplificada!

Este guia rápido vai te mostrar como é fácil integrar a Arara API e enviar sua primeira mensagem de template via WhatsApp. Nossa missão é descomplicar, e você verá isso desde o primeiro `curl`.

<Card icon="bolt" title="5 Minutos para o Sucesso">
  Siga os passos abaixo e tenha sua primeira mensagem enviada!!!
</Card>

## Pré-requisitos

Antes de começar, certifique-se de ter:

-   Uma **conta na Arara API** e acesso ao seu [Dashboard](https://dashboard.ararahq.com).
-   Uma **API Key de Teste (`sk_test_***`)**, que você pode gerar no seu [Dashboard em "Chaves de API"](https://dashboard.ararahq.com/api-keys).
-   Um número de celular **verificado** na sua conta Arara para receber as mensagens de teste.
-   Um cliente HTTP (como `curl` no terminal, Postman, Insomnia ou VS Code com extensão REST Client).

## Passo a Passo: Sua Primeira Mensagem

<Steps>
  ### Passo 1: Obtenha sua API Key
  <Step title="Gere sua API Key de Teste">
    Acesse o [Dashboard da Arara API](https://dashboard.ararahq.com) e navegue até a seção "Chaves de API". Gere uma nova chave de teste (`sk_test_***`) e **guarde-a em um local seguro**.

    <Info>
      Sua API Key deve ser tratada como uma senha. **Nunca a exponha em código cliente ou repositórios públicos.**
    </Info>
  </Step>

  ### Passo 2: Prepare o Template
  <Step title="Use um Template Padrão">
    Para este exemplo rápido, vamos usar um template de teste genérico que já está pré-aprovado na sua conta de **Dev Mode**.

    **Nome do Template:** `hello_world`

    Este template geralmente tem um único parâmetro, por exemplo: `Olá {{1}}! Bem-vindo à Arara API.`.
  </Step>

  ### Passo 3: Faça a Chamada da API
  <Step title="Envie a Mensagem com cURL">
    Abra seu terminal ou cliente HTTP e faça uma requisição `POST` para o endpoint de envio de mensagens.

    Substitua:
    -   `<SUA_API_KEY>` pela sua chave de teste (`sk_test_***`).
    -   `<NUMERO_VERIFICADO>` pelo número de celular que você verificou na sua conta Arara.

    ```bash curl
    curl -X POST [https://api.ararahq.com/v1/messages/send](https://api.ararahq.com/v1/messages/send) \
         -H "Authorization: Bearer <SUA_API_KEY>" \
         -H "Content-Type: application/json" \
         -d '{
               "to": "<NUMERO_VERIFICADO>",
               "template_name": "hello_world",
               "template_params": ["Usuário Arara"]
             }'
    ```
    
    <Card icon="python" title="Exemplo em Python (Opcional)">
      ```python
      import requests

      api_key = "<SUA_API_KEY>"
      to_number = "<NUMERO_VERIFICADO>"
      
      headers = {
          "Authorization": f"Bearer {api_key}",
          "Content-Type": "application/json"
      }
      
      payload = {
          "to": to_number,
          "template_name": "hello_world",
          "template_params": ["Usuário Arara"]
      }
      
      response = requests.post("[https://api.ararahq.com/v1/messages/send](https://api.ararahq.com/v1/messages/send)", headers=headers, json=payload)
      
      print(response.json())
      ```
    </Card>
  </Step>

  ### Passo 4: Verifique a Resposta e os Logs
  <Step title="Confirme o Envio">
    Se a requisição for bem-sucedida, você receberá uma resposta JSON similar a esta:
    ```json
    {
      "message_id": "msg_xyz123abc456",
      "status": "queued",
      "to": "<NUMERO_VERIFICADO>"
    }
    ```
    
    Além disso, você pode verificar o status e o histórico completo da mensagem no seu [Dashboard em "Logs"](https://dashboard.ararahq.com/logs).
  </Step>
</Steps>
