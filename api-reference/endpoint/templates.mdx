A API de Templates da Arara permite que você crie, liste e monitore o status de aprovação dos seus modelos de mensagem. Nossa implementação é focada em **Developer Experience**, abstraindo a complexidade da Meta e garantindo que seus templates sejam aprovados com validações inteligentes (Foolproof).

<Warning>
  A API de Templates é exclusiva para o **Modo Produção (LIVE)**. Requisições em modo de teste retornarão erro 403.
</Warning>

## Criar um Template `POST /v1/templates`

Este endpoint cria um template na infraestrutura da Arara e o submete automaticamente para aprovação da Meta.

### Exemplos de Uso

Para facilitar a integração, preparamos exemplos "copia e cola" para os cenários mais comuns.

<AccordionGroup>
  <Accordion title="Texto Simples (Básico)">
    Ideal para notificações rápidas e diretas.
    
    ```json
    {
      "name": "notificacao_simples",
      "category": "UTILITY",
      "language": "pt_BR",
      "body": "Olá {{1}}, seu agendamento foi confirmado para {{2}}.",
      "samples": {
        "1": "Maria",
        "2": "14/05 às 15h"
      }
    }
    ```
  </Accordion>

  <Accordion title="Com Header de Mídia (Imagem/PDF)">
    Inicie a conversa com um impacto visual ou envie faturas automaticamente.
    
    ```json
    {
      "name": "fatura_disponivel",
      "category": "UTILITY",
      "language": "pt_BR",
      "header": "https://ararahq.com/l/ FtFmja",
      "headerType": "media",
      "body": "Olá {{1}}, sua fatura do mês {{2}} já está disponível.",
      "samples": {
        "1": "João",
        "2": "Janeiro"
      }
    }
    ```
  </Accordion>

  <Accordion title="Botões Quick Reply (Resposta Rápida)">
    Permita que o usuário responda com apenas um toque. Máximo de 3 botões.
    
    ```json
    {
      "name": "pesquisa_satisfacao",
      "category": "MARKETING",
      "language": "pt_BR",
      "body": "Olá! O que achou do nosso atendimento?",
      "buttons": [
        { "type": "QUICK_REPLY", "text": "Amei! 😍" },
        { "type": "QUICK_REPLY", "text": "Foi bom 🙂" },
        { "type": "QUICK_REPLY", "text": "Pode melhorar 😐" }
      ]
    }
    ```
  </Accordion>

  <Accordion title="Call to Action (Link + Telefone)">
    Direcione o usuário para uma ação externa.
    
    ```json
    {
      "name": "recuperacao_carrinho",
      "category": "MARKETING",
      "language": "pt_BR",
      "body": "Ei {{1}}, você esqueceu itens no carrinho! Volte agora e ganhe 5% OFF.",
      "buttons": [
        { 
          "type": "URL", 
          "text": "Finalizar Compra", 
          "url": "https://loja.com/carrinho" 
        },
        { 
          "type": "PHONE_NUMBER", 
          "text": "Ligar para Vendedor", 
          "phone": "+5511999999999" 
        }
      ],
      "samples": { "1": "Ana" }
    }
    ```
  </Accordion>

  <Accordion title="🏆 Smart Link (Rastreia Cliques)">
    A "Killer Feature" da Arara. Encurta e rastreia quem clicou no link.
    
    ```json
    {
      "name": "oferta_relampago",
      "category": "MARKETING",
      "language": "pt_BR",
      "body": "Oferta imperdível para você, {{1}}!",
      "buttons": [
        { 
          "type": "SMART_LINK", 
          "text": "Ver Oferta Exclusiva",

        "2": "promocao-verao" 
      }
    }
    ```
  </Accordion>

### Regras de Ouro (Anti-Rejeição)

Para garantir que seu template seja aprovado rapidamente, aplicamos as seguintes regras:

1.  **Cabeçalhos Dinâmicos**: Se usar `headerType: "text"`, o cabeçalho deve ser estático ou conter apenas uma variável `{{1}}`. Se usar `media`, o `header` deve ser uma URL válida de uma imagem (JPG/PNG) ou PDF.
2.  **Normalização de Telefone**: No botão `PHONE_NUMBER`, a Arara normaliza automaticamente o número para o formato **E.164**. Você pode enviar `(11) 99999-9999` e nós converteremos para `+5511999999999`.
3.  **Botões**: Você pode usar até 3 botões de **Quick Reply** OU 1 botão de **URL/Smart Link** + 1 botão de **Telefone**. Não é permitido misturar tipos.
4.  **Smart Links**: Utilize o tipo `SMART_LINK` em botões de URL para que a Arara encurte e rastreie os cliques automaticamente.

## Listar Templates `GET /v1/templates`

Retorna todos os templates criados pela sua organização.

| Parâmetro | Tipo | Descrição |
| :--- | :--- | :--- |
| `name` | query | (Opcional) Filtra um template específico pelo nome exato. |

### Resposta

Retorna uma lista de objetos contendo o `providerStatus` (ex: `PENDING`, `APPROVED`, `REJECTED`) e o `variablesSchema` para ajudar no envio de mensagens.

## Consultar Status em Tempo Real `GET /v1/templates/{id}/status`

Embora a Arara atualize o status automaticamente via Webhook, você pode consultar manualmente o estado atual na Meta.

**Resposta em caso de rejeição:**
```json
{
  "status": "REJECTED",
  "rejectionReason": "INCORRECT_CATEGORY",
  "category": "MARKETING"
}
```

## Deletar Template `DELETE /v1/templates/{id}`

Remove o template da base da Arara. 

<Info>
  A remoção na Arara não remove automaticamente o template fisicamente da conta WABA na Meta por questões de histórico, mas impede novos envios através deste ID.
</Info>
