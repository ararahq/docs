## Por que essa página existe

A Arara está construindo um **Flow Builder visual** — desenhe orquestrações multi-step (template inicial → espera 2h → se não respondeu manda lembrete → se respondeu chama Brain) sem código. Quando lançar, esta página vai documentar a API REST de disparo.

## API planejada (sujeita a mudança)

A interface está sendo desenhada nestas linhas:

```bash
# Disparar flow publicado pra um contato
POST /v1/flows/{flowId}/trigger
{
  "phone": "+5511999999999",
  "metadata": { "orderId": "12345" }
}

# Testar flow (dry-run)
POST /v1/flows/{flowId}/test
{
  "phone": "+5511999999999",
  "metadata": { "orderId": "12345" }
}

# Listar execuções
GET /v1/flows/{flowId}/executions
```

Não use ainda — esses endpoints **retornam 404** no backend atual.
