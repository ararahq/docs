---
title: 'Campanhas'
description: 'Dispare o mesmo template para milhares de contatos de uma vez.'
---

Campanhas sao o jeito certo de disparar em massa. Voce manda uma lista de contatos, a Arara valida o saldo, enfileira tudo e processa em background.

**Base URL:** `https://new-api.ararahq.com/api/v2`

<Note>
  **1 por 1:** use [`POST /v2/messages`](/api-reference/endpoint/v2/messages).
  **Em massa:** use `POST /v2/campaigns`.
</Note>

---

## Criar uma campanha

`POST /v2/campaigns`

O header `Idempotency-Key` e **obrigatorio** — ele garante que um retry nunca crie duas campanhas iguais.

<CodeGroup>

```bash cURL
curl -X POST https://new-api.ararahq.com/api/v2/campaigns \
  -H "Authorization: Bearer ara_live_xxx" \
  -H "Idempotency-Key: 7f8a1c2d-4e5f-6789-abcd-ef0123456789" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Black Friday 2026",
    "template": "promo_black_friday",
    "contacts": [
      { "to": "5511999999999", "variables": ["Joao"] },
      { "to": "5511888888888", "variables": ["Maria"] },
      { "to": "5511777777777", "variables": ["Carlos"] }
    ]
  }'
```

```javascript Node.js
import { randomUUID } from "crypto";

await fetch("https://new-api.ararahq.com/api/v2/campaigns", {
  method: "POST",
  headers: {
    "Authorization": "Bearer ara_live_xxx",
    "Idempotency-Key": randomUUID(),
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Black Friday 2026",
    template: "promo_black_friday",
    contacts: [
      { to: "5511999999999", variables: ["Joao"] },
      { to: "5511888888888", variables: ["Maria"] },
      { to: "5511777777777", variables: ["Carlos"] },
    ],
  }),
});
```

```python Python
import requests, uuid

requests.post(
    "https://new-api.ararahq.com/api/v2/campaigns",
    headers={
        "Authorization": "Bearer ara_live_xxx",
        "Idempotency-Key": str(uuid.uuid4()),
    },
    json={
        "name": "Black Friday 2026",
        "template": "promo_black_friday",
        "contacts": [
            {"to": "5511999999999", "variables": ["Joao"]},
            {"to": "5511888888888", "variables": ["Maria"]},
            {"to": "5511777777777", "variables": ["Carlos"]},
        ],
    },
)
```

</CodeGroup>

Resposta (`201 Created`):

```json
{
  "id": "ara_cmp_7b3789fb",
  "name": "Black Friday 2026",
  "template": "promo_black_friday",
  "status": "PROCESSING",
  "totalRecipients": 3,
  "sent": 0,
  "delivered": 0,
  "failed": 0,
  "cost": 0.48,
  "createdAt": "2026-04-13T18:30:00Z",
  "completedAt": null
}
```

### Parametros

| Campo | Tipo | Obrigatorio | Descricao |
|-------|------|-------------|-----------|
| `name` | string | sim | Nome identificador da campanha. |
| `template` | string | sim | Nome do template pre-aprovado. |
| `contacts` | array | sim | Lista de destinatarios. Nao pode ser vazia. |
| `contacts[].to` | string | sim | WhatsApp: E.164 sem `+` (`5511999999999`). |
| `contacts[].variables` | string[] | nao | Array ordenado dos valores dos placeholders `{{1}}`, `{{2}}`... |

<Note>
  Idempotency-Key e **obrigatorio** no header. Gere um UUID novo para cada campanha.
</Note>

## Estimar custo antes de disparar

`GET /v2/campaigns/estimate`

Calcule o custo sem criar a campanha. Util pra checar saldo ou mostrar o total ao usuario antes de confirmar.

```bash
curl "https://new-api.ararahq.com/api/v2/campaigns/estimate?templateName=promo_black_friday&count=50000" \
  -H "Authorization: Bearer ara_live_xxx"
```

### Query params

| Param | Tipo | Descricao |
|-------|------|-----------|
| `templateName` | string | Nome do template. |
| `count` | integer | Quantidade de destinatarios. |

Resposta (`200 OK`):

```json
{
  "templateCost": 0.04,
  "araraFee": 0.12,
  "unitPrice": 0.16,
  "totalCost": 8000.00,
  "templateCategory": "UTILITY",
  "recipientCount": 50000
}
```

| Campo | Descricao |
|-------|-----------|
| `templateCost` | Custo por mensagem cobrado pelo provider (Meta). |
| `araraFee` | Taxa da Arara por mensagem. |
| `unitPrice` | `templateCost + araraFee`. |
| `totalCost` | `unitPrice * recipientCount`. |
| `templateCategory` | `UTILITY`, `MARKETING` ou `AUTHENTICATION`. Afeta o custo. |

## Consultar uma campanha

`GET /v2/campaigns/{id}`

```bash
curl https://new-api.ararahq.com/api/v2/campaigns/ara_cmp_7b3789fb \
  -H "Authorization: Bearer ara_live_xxx"
```

Resposta (`200 OK`):

```json
{
  "id": "ara_cmp_7b3789fb",
  "name": "Black Friday 2026",
  "template": "promo_black_friday",
  "status": "COMPLETED",
  "totalRecipients": 50000,
  "sent": 48500,
  "delivered": 47200,
  "failed": 1500,
  "cost": 7760.00,
  "createdAt": "2026-04-13T18:30:00Z",
  "completedAt": "2026-04-13T19:10:00Z"
}
```

### Campos

| Campo | Tipo | Descricao |
|-------|------|-----------|
| `id` | string | ID da campanha (`ara_cmp_...`). |
| `name` | string | Nome informado na criacao. |
| `template` | string | Template usado. |
| `status` | string | Status atual (ver abaixo). |
| `totalRecipients` | int | Total de contatos na campanha. |
| `sent` | int | Quantos foram enviados ao provider. |
| `delivered` | int | Quantos chegaram ao destinatario. |
| `failed` | int | Quantos falharam definitivamente. |
| `cost` | number | Custo real acumulado em BRL. |
| `createdAt` | string | Quando a campanha foi criada. |
| `completedAt` | string \| null | Quando terminou. `null` enquanto em processamento. |

## Listar campanhas

`GET /v2/campaigns`

```bash
curl "https://new-api.ararahq.com/api/v2/campaigns?page=0&size=20" \
  -H "Authorization: Bearer ara_live_xxx"
```

### Query params

| Param | Tipo | Default | Descricao |
|-------|------|---------|-----------|
| `page` | int | `0` | Pagina (base 0). |
| `size` | int | `20` | Itens por pagina. |

Resposta (`200 OK`):

```json
{
  "data": [
    {
      "id": "ara_cmp_7b3789fb",
      "name": "Black Friday 2026",
      "template": "promo_black_friday",
      "status": "COMPLETED",
      "totalRecipients": 50000,
      "sent": 48500,
      "delivered": 47200,
      "failed": 1500,
      "cost": 7760.00,
      "createdAt": "2026-04-13T18:30:00Z",
      "completedAt": "2026-04-13T19:10:00Z"
    }
  ],
  "pagination": {
    "page": 0,
    "size": 20,
    "totalElements": 1,
    "totalPages": 1
  }
}
```

## Status

| Status | Significado |
|--------|-------------|
| `PROCESSING` | Contatos sendo enfileirados para envio. |
| `COMPLETED` | Todos os contatos foram processados. |
| `FAILED` | Falha geral (ex: saldo insuficiente antes de iniciar). |

Para acompanhar em tempo real a entrega de cada mensagem individual, assine os [webhooks](/api-reference/events) de `message.delivered` / `message.failed`.

---

## Erros

| HTTP | Quando |
|------|--------|
| `400` | `contacts` vazio, `name`/`template` em branco. |
| `401` | Chave invalida. |
| `402` | Saldo insuficiente para cobrir a campanha. |
| `404` | Template nao encontrado, campanha inexistente (no GET). |
| `409` | Idempotency-Key ja usada. |
| `429` | Rate limit excedido. |
