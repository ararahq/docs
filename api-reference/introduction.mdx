Bem-vindo à central de comando da Arara. Nossa API v1 foi meticulosamente desenhada para ser a ponte mais rápida, robusta e escalável entre sua aplicação e o ecossistema WhatsApp Business.

## A Filosofia Arara: Engineering First

A Arara nasceu para eliminar a fricção entre o código e o cliente. Nossos diferenciais são:
- **Simplicidade Radical**: Endpoints previsíveis, RESTful e sem boilerplate.
- **Escala Massiva**: Infraestrutura testada para milhões de mensagens com Lazy-Time Queue.
- **Diagnóstico Humano**: O **Escudo de Erros** traduz falhas da Meta em soluções acionáveis.

## Base URL

Todas as requisições à API v1 devem ser feitas para:

```http
https://api.ararahq.com/api/v1
```

### Base URL V2

<Note>
  **Migrando para a V2?** Se voce esta utilizando a nova infraestrutura V2 da AraraHQ, sua base URL sera diferente. Substitua a URL acima por:

  ```http
  https://new-api.ararahq.com/api/v1
  ```

  Nos SDKs oficiais, basta configurar o parametro `baseUrl` (Node.js) ou `base_url` (Python) ao instanciar o cliente. Via variavel de ambiente, defina `ARARA_BASE_URL=https://new-api.ararahq.com/api`.

  A API V1 (`api.ararahq.com`) continua funcionando normalmente. A migracao para V2 e opcional e pode ser feita gradualmente.
</Note>

## Autenticação

Segurança é nossa prioridade. Autentique suas chamadas usando o cabeçalho `Authorization` com sua **API Key**:

```bash
Authorization: Bearer ara_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Padrões Globais

### Formato de Resposta
A API responde exclusivamente em **JSON**. Todas as datas seguem o padrão **ISO 8601 (UTC)**.

### Códigos de Status HTTP

| Código | Descrição | Contexto Arara |
|--------|-----------|---------------|
| `200` | OK | Sucesso síncrono (ex: listar templates). |
| `202` | Accepted | Requisição aceita para processamento (ex: envio de mensagem). |
| `400` | Bad Request | Parâmetros inválidos ou malformados. |
| `401` | Unauthorized | API Key inválida ou ausente. |
| `422` | Unprocessable Entity | **Escudo de Erros**: Dados válidos, mas regra de negócio violada. |
| `500` | Server Error | Falha interna no processamento Arara. |

## Boas Práticas e Compliance

A Arara preza pela qualidade da rede WhatsApp. Para manter sua pontuação de qualidade alta:
- ✅ **Opt-in Explicito**: Envie mensagens apenas para clientes que autorizaram.
- ✅ **Conteúdo Relevante**: Evite spam e mensagens excessivamente comerciais.
- ✅ **Tratamento de Erros**: Implemente lógica de retry baseada nos diagnósticos do nosso Escudo de Erros.
