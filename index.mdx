# Arara: Descomplicando o WhatsApp no Brasil

Sua jornada para integrar a comunicação via WhatsApp nunca foi tão simples e eficiente. A Arara é a plataforma que abstrai toda a complexidade da Meta, permitindo que você envie mensagens de template em **menos de 5 minutos**.

## Visão Geral

A Arara é a solução ideal para desenvolvedores, startups de SaaS e agências digitais no Brasil que buscam **simplicidade radical** e uma **Experiência do Desenvolvedor (DX) de primeira classe**.

- **Para Quem?** Desenvolvedores, Startups de SaaS, Agências Digitais.
- **Por Quê?** Eliminar a frustração da API oficial do WhatsApp com uma solução focada no mercado brasileiro.
- **O Quê?** Uma API para envio de mensagens de template via WhatsApp com templates gerenciados visualmente e webhooks para integração com carrinhos de compra.

## Como Funciona

A Arara funciona de forma simples e direta:

1. **Autenticação**: Faça login no dashboard e receba sua chave de API única (mostrada apenas uma vez)
2. **Templates**: Crie seus templates de mensagem diretamente no dashboard
3. **Envio**: Use nossa API REST para enviar mensagens usando os templates criados
4. **Webhooks**: Configure webhooks para receber notificações de eventos, como atualizações de carrinho
