# SDKs e Bibliotecas

Para acelerar sua integração, fornecemos SDKs oficiais e bibliotecas mantidas pela comunidade. Nossos SDKs seguem padrões de **Clean Code** e oferecem suporte nativo a tipos (TypeScript/Kotlin/Python).

## Por Que Usar Nossos SDKs?

Nossos SDKs não são apenas "wrappers" de API. Eles incluem:

1.  **Retentativas Automáticas**: Lógica de *Exponential Backoff* embutida para lidar com instabilidades de rede.
2.  **Validação Local**: Verificamos o formato dos números e nomes dos templates antes mesmo de fazer o request, economizando tempo de latência.
3.  **Tipagem Forte**: Autocomplete completo no seu editor (VS Code, IntelliJ) para todos os payloads e respostas.
4.  **Logging Integrado**: Facilita o debug mapeando erros da API diretamente para exceções na sua linguagem.

## Exemplo de Uso (TypeScript)

```typescript
import { AraraClient } from '@ararahq/sdk';

const arara = new AraraClient('ara_sk_...');

const message = await arara.messages.send({
  receiver: 'whatsapp:+5583991768778',
  templateName: 'boas_vindas',
  variables: ['Micael']
});

console.log(`Mensagem enviada! ID: ${message.id}`);
```
