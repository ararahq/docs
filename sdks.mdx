## Código Aberto & Soberania

Todos os nossos SDKs são distribuídos sob a licença **MIT** e estão disponíveis no nosso [GitHub](https://github.com/ararahq). Acreditamos que a infraestrutura de mensagens deve ser transparente e extensível.

---

## Por Que Usar Nossos SDKs?

Nossos SDKs não são apenas "wrappers" de API. Eles incluem:

1.  **Retentativas Automáticas**: Lógica de *Exponential Backoff* embutida para lidar com instabilidades.
2.  **Validação Local**: Verificamos o formato dos números e nomes dos templates antes do request.
3.  **Tipagem Forte**: Autocomplete completo no seu editor (VS Code, IntelliJ) para todos os payloads.
4.  **Universal Webhooks**: Tipagem nativa para processar eventos de status e mensagens recebidas.

## Exemplo de Uso Multi-Linguagem

Escolha sua stack e veja como é simples enviar uma mensagem com mídia e variáveis.

<CodeGroup>
```typescript Node.js
import { NodeSDK } from '@ararahq/sdk';

const sdk = new NodeSDK({ apiKey: 'sk_live_...' });

const response = await sdk.messages.send({
  receiver: 'whatsapp:+5511999998888',
  templateName: 'fatura_disponivel',
  variables: ['João', 'Janeiro'],
  media_url: 'https://ararahq.com/l/FtFmja'
});
```

```php PHP
use Arara\Sdk\Arara;

$arara = new Arara('sk_live_...');

$response = $arara->messages()->send([
    'receiver' => 'whatsapp:+5511999998888',
    'templateName' => 'fatura_disponivel',
    'variables' => ['João', 'Janeiro'],
    'media_url' => 'https://ararahq.com/l/FtFmja'
]);
```

```python Python
from arara_api_sdk import AraraClient

with AraraClient(api_key="sk_live_...") as client:
    response = client.messages.send({
        "receiver": "whatsapp:+5511999998888",
        "templateName": "fatura_disponivel",
        "variables": ["João", "Janeiro"],
        "media_url": "https://ararahq.com/l/FtFmja"
    })
```

```java Java
Arara arara = Arara.builder().apiKey("sk_live_...").build();

MessageResponse response = arara.getMessages().send(
    SendMessageRequest.builder()
        .receiver("whatsapp:+5511999998888")
        .templateName("fatura_disponivel")
        .templateVariables(List.of("João", "Janeiro"))
        .mediaUrl("https://ararahq.com/l/FtFmja")
        .build()
);
```
</CodeGroup>
