
## Por que usar nossos SDKs

Os SDKs não são wrappers fininhos da API. Cada um inclui:

- **Retry com backoff exponencial** pra 429 e 5xx
- **Validação local** de número (E.164), template name, schema
- **Tipos nativos** pra todos os request/response (TypeScript, Pydantic, Lombok)
- **Tipos pra webhook** (`AraraWebhookEvent` discriminated union)
- **Idempotency-Key** integrado
- **Headers de auth** automáticos
- **Honra `Retry-After`** em 429

## Exemplo lado a lado

Mesma operação (enviar template com header de mídia + 2 variáveis):

<CodeGroup>
```typescript Node.js
import { NodeSDK } from '@ararahq/sdk';

const sdk = new NodeSDK({
  apiKey: process.env.ARARA_API_KEY!,
  baseUrl: 'https://api.ararahq.com',
});

await sdk.messages.send({
  receiver: 'whatsapp:+5511999999999',
  templateName: 'fatura_disponivel',
  templateVariables: [
    'https://meusite.com/fatura.pdf', // header (mídia)
    'João',                            // {{1}}
    'Janeiro'                          // {{2}}
  ],
});
```

```python Python
from arara_api_sdk import AraraClient
from arara_api_sdk.models.message import SendMessageRequest

with AraraClient(api_key="ara_live_xxx") as client:
    client.messages.send(SendMessageRequest(
        receiver="whatsapp:+5511999999999",
        template_name="fatura_disponivel",
        template_variables=[
            "https://meusite.com/fatura.pdf",
            "João",
            "Janeiro",
        ],
    ))
```

```php PHP
use Arara\Arara;
use Arara\Config;

$sdk = new Arara(new Config(apiKey: 'ara_live_xxx'));

$sdk->messages->send(
    receiver: 'whatsapp:+5511999999999',
    templateName: 'fatura_disponivel',
    variables: [
        'https://meusite.com/fatura.pdf',
        'João',
        'Janeiro',
    ],
);
```

```java Java
import com.ararahq.arara.sdk.Arara;
import com.ararahq.arara.sdk.models.SendMessageRequest;
import java.util.List;

Arara arara = Arara.builder().apiKey("ara_live_xxx").build();

arara.getMessages().send(
    SendMessageRequest.builder()
        .receiver("whatsapp:+5511999999999")
        .templateName("fatura_disponivel")
        .templateVariables(List.of(
            "https://meusite.com/fatura.pdf",
            "João",
            "Janeiro"
        ))
        .build()
);
```
</CodeGroup>

**Padrão de mídia**: URL da mídia (imagem/vídeo/PDF) vai como **primeiro elemento** do array `templateVariables`. Posições seguintes preenchem `{{1}}`, `{{2}}`... do corpo do template.
