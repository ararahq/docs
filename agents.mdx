A Arara expõe isso por 3 superfícies:

## O que faz a diferença

```
Agente puro                    Agente + Arara
"Quer fazer pedido?"           "Quer fazer pedido? Aqui está o Pix de R$ 47."
"Quando posso atender?"        "Tenho quarta 10h e quinta 14h. Confirmou pra quarta."
"Tem essa camisa em M?"        "Sim, R$ 79,90, restam 8. Aqui o checkout."
"Vou repassar pro atendente."  *Resolveu sozinho. Avisa humano só se urgência subir.*
```

A diferença é **executar**, não só conversar.

## Guardian Mode — segurança da marca

Todo agente que envia mensagem pela Arara passa pelo **Guardian Mode** antes de qualquer envio. O Guardian roda regras (PII, dados sensíveis, palavras proibidas, tom inadequado pra marca) e bloqueia o envio se detectar problema — sem você precisar instrumentar nada.

```json
{
  "blockOnPII": ["cpf", "rg", "credit_card"],
  "blockKeywords": ["pague aqui", "última chance"],
  "requireApprovalAbove": 1000,
  "brandTone": "informal, sem emojis em excesso"
}
```

Configurável por sessão via `configure_guardian_policy` (tool MCP) ou no dashboard.

## Tools nativas, não Zapier

Diferença filosófica: Zapier é **integração externa que cola APIs**. As tools da Arara são **nativas** — rodam no nosso backend, com auth/idempotência/retry/logs/Guardian. Tudo:

- Sob seus secrets, criptografados em repouso
- Sem latência de hop extra
- Com observabilidade direta no dashboard da Arara
- Sob a mesma SLA da sua conta

Não substitui Zapier pra automações genéricas. Substitui pra tudo que toca WhatsApp.
