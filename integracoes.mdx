A Arara foi construída pra encaixar onde você já está. Não tem framework próprio, não tem GUI proprietária. Só API, SDKs e Webhooks — o resto é integração nativa com o que você usa hoje.

## Sua stack não tá aqui?

A API da Arara é REST padrão. Qualquer linguagem com HTTP client funciona. [Veja a referência completa](/api-reference/introduction) ou abra uma issue no [GitHub](https://github.com/ararahq) se quiser uma integração nativa específica.
