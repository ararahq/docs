A regra geral: **3 atores cobram**. Esta página mostra cada um com número real em BRL.

```
Meta (Cloud API) cobra por mensagem entregue
       ↓
Arara repassa o custo Meta + acréscimo fixo por mensagem
       (R$ 0,25 no Free · R$ 0,10 no Voo)
       ↓
Você fatura pro cliente final (opcional)
```

<Note>
  Os preços abaixo são **fixos em BRL** pra números brasileiros — o custo Meta já vem convertido e embutido, sua fatura não flutua com câmbio. Números internacionais têm custo Meta diferente por país; use a [estimativa via API](#estimativa-via-api).
</Note>

## Custo por mensagem (categoria)

Meta cobra **por mensagem entregue**. Falhas (template rejeitado, número inválido, fora da janela) não custam.

| Categoria | Free | Voo |
|---|---|---|
| **MARKETING** — promo, cupom, broadcast | R$ 0,65 | **R$ 0,45** |
| **UTILITY** — pedido, boleto, agendamento | R$ 0,30 | **R$ 0,15** |
| **AUTHENTICATION** — OTP, 2FA, verificação | R$ 0,30 | **R$ 0,15** |
| **Sessão (janela 24h)** — você respondendo | R$ 0,25 | **R$ 0,10** |

Na **janela de 24h** (cliente te chamou, você responde texto livre) a Meta não cobra — você paga só o acréscimo Arara. Fora da janela, só template aprovado, com o custo Meta da categoria embutido.

## Planos

Dois planos. Sem tier intermediário, sem "fale com vendas".

| | Free | Voo |
|---|---|---|
| **Assinatura** | R$ 0 | R$ 349/mês |
| **Pra quem** | Dev integrando por API | Operação completa |
| **Número WhatsApp** | 1 próprio (BYON) | Incluído, provisionado pela Arara |
| **API + SDKs + webhooks** | Sim | Sim |
| **Mensagens e templates** | Sem cap | Sem cap |
| **Campanhas, inbox, contatos, lembretes, Brain** | — | Sim |
| **Mensagem** | Meta + R$ 0,25 | Meta + R$ 0,10 |

Saldo de mensagens é **wallet pré-paga** (Pix ou cartão) e não expira. Quem assinou os planos antigos (Essential, Pro, Business) segue nas condições contratadas — essa tabela vale pra contas novas.

## Calculadora

Cenário 1 — **E-commerce médio** (Voo):
- 8.000 mensagens UTILITY/mês (confirmações, rastreio)
- 4.000 mensagens MARKETING/mês (carrinho abandonado, campanhas)

```
Plano Voo:                                  R$   349
8.000 × R$ 0,15 (UTILITY):                  R$ 1.200
4.000 × R$ 0,45 (MARKETING):                R$ 1.800
                                           ─────────
Fatura mensal:                              R$ 3.349
```

Cenário 2 — **SaaS B2B** (Voo):
- 1.500 mensagens AUTHENTICATION/mês (OTP login)
- 500 mensagens UTILITY/mês (alertas)

```
Plano Voo:                                  R$   349
1.500 × R$ 0,15 (AUTH):                     R$   225
500 × R$ 0,15 (UTILITY):                    R$    75
                                           ─────────
Fatura mensal:                              R$   649
```

Cenário 3 — **Dev prototipando** (Free):
- 300 mensagens UTILITY/mês em teste de integração

```
Assinatura:                                 R$     0
300 × R$ 0,30 (UTILITY):                    R$    90
                                           ─────────
Fatura mensal:                              R$    90
```

O delta de mensagem entre Free e Voo (R$ 0,15 na UTILITY/AUTH, R$ 0,20 no MARKETING) paga o plano sozinho a partir de ~2.000 mensagens/mês. Acima disso, Voo é mais barato **e** inclui a operação inteira.

## Estimativa via API

```bash
curl "https://api.ararahq.com/api/v1/wallet/calculate?amountCents=10000" \
  -H "Authorization: Bearer ara_live_xxx"
```

Retorna quantas mensagens cada R$ 100 compra por categoria + janela 24h, já no preço do seu plano.

## Comparativo

### vs SMS BR

| | SMS | WhatsApp via Arara |
|---|---|---|
| Custo por envio | R$ 0,12 - 0,15 (entregue ou não) | R$ 0,15 - 0,65 (só entregue) |
| Taxa de abertura | ~50% | ~98% |
| CTR | ~1% | ~8-10% |
| Resposta possível | Não | Sim (janela 24h) |
| **CAC efetivo** | Alto (poucos convertem) | Baixo (alto CTR) |

WhatsApp custa 1-4x mais por envio, converte 8-10x mais. CAC efetivo menor.

### vs Z-API / gateway pirata

| | Gateway pirata ("grátis") | Arara |
|---|---|---|
| Custo aparente | R$ 100-300/mês fixo | Free + uso, ou R$ 349 tudo incluso |
| Custo real (incluindo banimento) | **R$ X.000 quando o número quebra** + 30 dias parado | Previsível |
| Risco de banimento | Alto | Quase zero |
| Templates aprovados | Não tem | Sim |
| Volume máximo | 200-500 msg/dia | Sem limite real (com warming) |

## Onde ver fatura

Dashboard → **Cobrança → Faturas**. Cada fatura tem:

- Resumo: plano + créditos consumidos + saldo
- Detalhamento por categoria
- Cohort por template (quanto cada template custou)

## Wallet e auto-recharge

Saldo é em **créditos pré-pagos** (R$ depositados via Pix ou cartão). Quando saldo cai abaixo do threshold, auto-recharge dispara cartão e completa.

```bash
curl https://api.ararahq.com/api/v1/wallet/auto-recharge \
  -H "Authorization: Bearer ara_live_xxx"
```

Configuração: dashboard → **Cobrança → Auto-recarga** ou via PATCH no endpoint acima.
