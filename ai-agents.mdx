## Os 5 Pilares do Revenue OS

O nosso MCP foi construído sobre cinco pilares fundamentais para escala:

<Steps>
  <Step title="Autonomous Revenue Recovery">
    A IA monitora "leaks" de receita (Pix vencidos, falhas de cartão) e sugere ações de recuperação imediatas.
  </Step>
  <Step title="Atomic Negotiation">
    Ferramentas para a IA negociar descontos e gerar links de pagamento do AbacatePay diretamente no chat.
  </Step>
  <Step title="Guardian Mode (Brand Safety)">
    Um firewall de IA que intercepta mensagens sensíveis ou fora de política antes do envio.
  </Step>
  <Step title="Business Memory Layer (BML)">
    Vetoraização do histórico para que a IA saiba o "mood" e o contexto de cada cliente.
  </Step>
  <Step title="Mass Orchestration">
    Disparos inteligentes e segmentados para milhares de contatos com gestão automatizada de respostas.
  </Step>
</Steps>

## Referência de Ferramentas (Tools)

Quando você conecta o MCP, as seguintes ferramentas ficam disponíveis para a IA:

| Ferramenta | O que faz? |
| :--- | :--- |
| `send_smart_message` | Envia mensagens via WhatsApp (integrado ao Guardian Mode). |
| `check_revenue_leaks` | **(V2)** Identifica cobranças pendentes ou vencidas no AbacatePay. |
| `negotiate_payment` | **(V2)** Cria produtos e links de pagamento dinâmicos para fechar vendas no chat. |
| `confirm_payment_handshake` | **(V2)** Verifica o status real de um pagamento (Trusted Handshake). |
| `get_customer_insights` | **(BML)** Cruza histórico do Arara com metadados do AbacatePay. |
| `mass_orchestration` | Orquestra disparos em massa com firewall de segurança de IA. |

## Como Conectar (Modo Hosted)

Use nosso servidor oficial para ter acesso imediato a todas essas ferramentas.

### 1. Claude Desktop
Adicione ao seu `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "ararahq": {
      "url": "https://mcp.ararahq.com/sse",
      "headers": {
        "X-Arara-Key": "SUA_CHAVE_ARARA_AQUI",
        "X-Abacate-Key": "SUA_CHAVE_ABACATE_AQUI"
      }
    }
  }
}
```

### 2. Cursor
Adicione em **Settings > Models > MCP Servers**:
- **Name**: AraraHQ
- **Type**: `SSE`
- **URL**: `https://mcp.ararahq.com/sse`

## Como Liberar para Outros Usuários

Se você deseja que seus clientes ou colaboradores usem o Arara MCP em suas próprias máquinas:

1.  **Chave de API**: O usuário deve gerar uma `API_KEY` no painel da Arara.
2.  **Configuração**: Forneça o trecho JSON acima, instruindo-os a substituir `SUA_CHAVE_ARARA_AQUI` pela chave que geraram.
3.  **Segurança**: O servidor em `mcp.ararahq.com` é compartilhado e não armazena chaves. Cada chamada de ferramenta utiliza a chave enviada no header da sessão.

## SDK vs MCP: Qual usar?

A Arara oferece duas formas de integrar inteligência ao seu fluxo. Você pode usar ambas simultaneamente:

| Versão | Quando usar? | Benefício |
| :--- | :--- | :--- |
| **Node SDK** (`@ararahq/sdk`) | Backends tradicionais, scripts e automações fixas. | Tipagem forte e controle total do fluxo programático. |
| **MCP Server** (`sse`) | Chatbots, Agentes de IA e ferramentas como Claude/Cursor. | Descoberta automática de ferramentas pela IA sem código extra. |

## Exemplo: Vercel AI SDK + Arara

Você pode integrar a Arara de duas formas no Vercel AI SDK:

### Opção A: Via Node SDK (Controle Manual)
Ideal para quando você quer definir exatamente quais parâmetros a IA pode ver.

```typescript
import { NodeSDK } from '@ararahq/sdk';
import { generateText, tool } from 'ai';
import { z } from 'zod';

const arara = new NodeSDK({ apiKey: process.env.ARARA_API_KEY });

const result = await generateText({
  model: openai('gpt-4o'),
  tools: {
    sendWhatsApp: tool({
      description: 'Envia uma mensagem de WhatsApp via Arara',
      parameters: z.object({ to: z.string(), text: z.string() }),
      execute: async ({ to, text }) => arara.messages.send({ receiver: to, body: text }),
    }),
  },
  prompt: 'Notifique o cliente João que o seu pedido está pronto.',
});
```

### Opção B: Via MCP (Plug & Play)
Ideal para dar à IA acesso a todo o ecossistema (Arara + AbacatePay) instantaneamente.

```typescript
import { ModelContextProtocolClient } from '@modelcontextprotocol/sdk/client';
// O MCP expõe automaticamente send_smart_message, negotiate_payment, etc.
// Use o cliente MCP para conectar sua aplicação Node ao servidor mcp.ararahq.com
```

## Business Memory Layer (BML)

O BML não é apenas um log; é a inteligência que permite à IA saber:
1.  **Contexto Financeiro**: O cliente tem pagamentos atrasados? (AbacatePay).
2.  **Histórico Emocional**: O cliente está satisfeito ou teve falhas técnicas recentes? (Arara Messages).
3.  **Janela de Oportunidade**: A janela de 24h está aberta para uma oferta "Trusted Handshake"?

Essa camada garante que seu agente de IA sempre tome a decisão mais assertiva de comunicação, economizando créditos e protegendo a saúde do seu número.
