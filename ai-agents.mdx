# Arara + IA: O Futuro da Comunicação

A Arara foi construída com foco em **Agentes de IA**. Nossa API é otimizada para ser lida e executada por modelos como GPT-4, Claude e Gemini, servindo como os "braços" de execução dos seus assistentes.

## Model Context Protocol (MCP)

O **Arara MCP** é a forma mais rápida de dar superpoderes ao seu Claude Desktop, Cursor ou qualquer IDE compatível. Com ele, a IA ganha ferramentas nativas para enviar mensagens, consultar histórico e gerar cobranças.

### Como Conectar (Modo Hosted)

Nós mantemos um servidor MCP oficial pronto para uso. Você só precisa apontar seu cliente para a nossa URL e fornecer sua chave de API.

#### 1. Claude Desktop
Adicione o seguinte ao seu `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "ararahq": {
      "url": "https://mcp.ararahq.com/sse",
      "headers": {
        "X-Arara-Key": "SUA_CHAVE_ARARA_AQUI"
      }
    }
  }
}
```

#### 2. Cursor
Vá em **Settings > Models > MCP Servers** e adicione um novo servidor:
- **Name**: AraraHQ
- **Type**: `SSE`
- **URL**: `https://mcp.ararahq.com/sse`

---

## Como Liberar para Outros Usuários

Se você deseja que seus clientes ou colaboradores usem o Arara MCP em suas próprias máquinas:

1.  **Chave de API**: O usuário deve gerar uma `API_KEY` no painel da Arara.
2.  **Configuração**: Forneça o trecho JSON acima, instruindo-os a substituir `SUA_CHAVE_ARARA_AQUI` pela chave que geraram.
3.  **Segurança**: O servidor em `mcp.ararahq.com` é compartilhado e não armazena chaves. Cada chamada de ferramenta utiliza a chave enviada no header da sessão.
